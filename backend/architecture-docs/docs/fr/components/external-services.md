# Intégrations de Services Externes

La plateforme est conçue pour s'intégrer avec divers services externes afin de fournir une fonctionnalité métier complète.

## Architecture d'Intégration

Tous les services externes se connectent via le load balancer **LBINTEGRATION** pour une gestion centralisée de la sécurité.
