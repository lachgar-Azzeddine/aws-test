# Couche Application

La couche application se compose des applications utilisateur final qui s'exécutent sur la plateforme.

## Produits

La plateforme peut déployer deux produits principaux :

### EServices

**EServices** est une plateforme complète d'e-gouvernement pour fournir des services aux citoyens.

### GCO (Gestion Courrier Opérationnel)

**GCO** est un portail de gestion des opérations pour les utilisateurs internes et back office.
