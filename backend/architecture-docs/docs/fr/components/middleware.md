# Couche Middleware

La couche middleware fournit les services de base requis pour que les applications fonctionnent. La plupart des composants sont déployés sur les clusters Kubernetes.

## Orchestration de Conteneurs

### Rancher Kubernetes Engine 2 (RKE2)

**RKE2** est une distribution Kubernetes certifiée qui alimente l'orchestration de conteneurs de la plateforme.

Pour les détails complets sur la couche middleware, veuillez consulter la [version anglaise](../en/components/middleware.md) ou le document ARCHITECTURE.md source.
